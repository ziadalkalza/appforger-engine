# Review Integration Risk Skill

Use this skill to evaluate integration risk before approval.

Check:
- official/trusted/community/custom classification;
- read/write capability;
- scopes and credentials;
- sensitive data exposure;
- data duplication;
- RAG/graph indexing risk;
- cost/hosting impact;
- maintenance owner;
- audit logging;
- rollback/removal path.

High-risk integrations require explicit user approval before code generation, execution, write actions, or persistent indexing.
