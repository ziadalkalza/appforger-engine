# Plan Custom Python Integration Skill

Use this skill only after the Integration Strategy Advisor has checked connector/MCP and built-in AppForger options.

Custom Python requires approval and a test plan.

The plan must include:
- why custom code is justified;
- authentication method and secret placement;
- API/version assumptions;
- pagination/rate-limit handling;
- sensitive-field exclusions;
- dry-run command;
- mock/sample input path;
- validation command;
- iteration warning.

Make clear that custom code may need testing and refinement to match real source behavior.
