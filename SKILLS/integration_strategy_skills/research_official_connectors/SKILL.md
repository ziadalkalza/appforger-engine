# Research Official Connectors Skill

Use this skill before recommending a custom integration for an external service.

Research for:
- official provider connectors;
- official MCP servers;
- trusted marketplace connectors;
- supported AI clients/tools;
- read/write capabilities;
- auth method and scopes;
- cost/plan requirements;
- enterprise/admin controls;
- maintenance status.

Record findings in `project-control/integrations/integration-decision-log.md` and in the integration decision packet.

If browsing is unavailable, state that connector availability must be verified before final approval.
