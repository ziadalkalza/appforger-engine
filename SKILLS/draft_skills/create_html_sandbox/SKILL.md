# Create HTML Sandbox

## Purpose

Create a quick HTML/CSS/JS prototype for exploration only.

## Required context

- `AGENTS.md`
- `DRAFT_AND_SANDBOX_MODE/README.md`
- `project-control/REGISTRIES/DRAFT_REGISTRY.md` when available
- relevant epic/feature/story if known

## Rules

- Do not modify approved baselines directly.
- Do not write draft artifacts into `appforge-engine/`.
- Keep raw draft files in `local-only/` or a clearly marked draft branch.
- Update draft registries when the user wants the draft tracked.
- Ask before promotion.

## Steps

1. Confirm sandbox purpose.
2. Create files under local-only/drafts/html-prototypes or local-only/sandbox/web-preview.
3. Include visible draft warning in the prototype.
4. Record draft metadata.
5. Explain promotion path.

## Output

HTML sandbox files and draft record.
