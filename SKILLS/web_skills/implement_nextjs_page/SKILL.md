# Skill: Implement Next.js Page

## Purpose
Guide Codex implementation of a Next.js route/page from an approved page packet.

## Required inputs
- route registry entry
- Figma/page baseline packet
- API contract
- environment config

## Steps
1. Read source-of-truth files.
2. Implement route, layout, metadata, loading/error states, and components.
3. Ensure server/client boundaries are correct.
4. Run lint/type/test/build checks.
5. Produce a done report.
