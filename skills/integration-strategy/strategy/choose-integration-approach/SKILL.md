# Choose Integration Approach Skill

Use this skill whenever the user asks to add or design an integration.

Steps:
1. Clarify the goal: live access, persistent RAG, graph relationships, write-back, automation, or custom business logic.
2. Research current official connector/plugin/MCP options if the environment supports browsing. If not, mark research as required.
3. Check Appforger built-in integrations and source pipelines.
4. Compare connector/MCP, built-in pipeline, direct API, custom Python, manual export/import, and hybrid approaches.
5. Evaluate security, cost, hosting, credentials, data duplication, and maintenance.
6. Recommend an approach.
7. Create or update an integration decision packet.

Do not generate custom Python code until this skill has produced a recommendation and the user has approved the custom-code path.
