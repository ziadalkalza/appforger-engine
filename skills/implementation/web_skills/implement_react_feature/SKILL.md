# Skill: Implement React Feature

## Purpose
Guide code-agent implementation of a React feature from approved AppForge packets.

## Required inputs
- code-agent web task packet
- approved design baseline or explicit mock-only instruction
- API contract or mock API rule
- web component mapping

## Rules
- Do not invent visual design.
- Do not skip accessibility states.
- Do not create backend code in the web repo unless the selected stack allows it.
- Run type checks/tests before done.
- Update task status and done report.
