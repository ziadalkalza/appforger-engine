# Skill: create code agent packet

## Purpose

Create a provider-agnostic code-agent packet for the selected code agent.

## Rules

- Route by capability, not brand.
- Use project-control registries for provider preferences and decisions.
- Respect privacy, cost, security, team, draft/sandbox, and approval policies.
- For unsuitable current providers, offer packet generation or draft-only mode.
- Record high-risk provider decisions.

## Output

A concise report plus any created/updated project-control artifacts.
