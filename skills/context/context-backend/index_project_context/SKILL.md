# Skill: Index Project Context

## Purpose
Index approved project-control memory into the selected context backend.

## Steps
1. Load context backend config.
2. Validate exclusions.
3. Index approved project-control files and summaries.
4. Write sync report.
5. Run freshness validation.
6. Update CONTEXT_INDEX_REGISTRY.md and CONTEXT_SYNC_REGISTRY.md.

## Evidence
- Sync report path
- indexed count
- excluded count
- freshness result
