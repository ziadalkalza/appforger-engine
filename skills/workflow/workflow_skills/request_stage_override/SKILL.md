# Skill: Request Stage Override

Use when the user wants to skip, substitute, reorder, import, or draft a stage.

Steps:
1. Identify requested override type.
2. Enter plan mode.
3. Run impact analysis.
4. Identify non-negotiable rules.
5. Recommend safer alternatives.
6. Ask for user approval.
7. Create override record if approved.
8. Update project-control registries.
