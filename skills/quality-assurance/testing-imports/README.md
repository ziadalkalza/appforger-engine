# Testing Skill Imports

This layer lets Appforger import external open-source testing skills, testing methodologies, QA checklists, framework-specific testing guides, and quality playbooks.

Imported testing skills extend Appforger. They do not replace Appforger completion evidence, security, privacy, or project-control rules.

External testing skills must be license-checked, reviewed, mapped to the project stack, approved, and activated before use.
