# Create Test Plan Skill

## Purpose
Create a feature/stage test plan from requirements, baselines, risks, and active testing skills.

## Required inputs
- Feature spec or stage scope
- Acceptance criteria
- Current design/API baselines if available
- Active platform tracks
- Active external testing skills if any

## Steps
1. Read `validation/testing/test-authoring-execution/test-plan-template.md`.
2. Identify required test types.
3. Generate draft test plan.
4. Link tests to feature/screen/API IDs.
5. Identify required evidence and commands.
6. Update project-control test registry.

## Output
A test plan in project-control, not in the reusable engine.
