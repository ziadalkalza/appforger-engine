# Import Testing Skill

## Purpose
Import an external open-source testing skill, QA checklist, or testing methodology into Appforger.

## Steps
1. Identify source and license.
2. Classify testing scope.
3. Extract useful rules and templates.
4. Map to Appforger stages and project stacks.
5. Record conflicts.
6. Ask user for approval before activation.
7. Update external testing skill registry.

## Output
Imported testing skill metadata and adapted usage notes in project-control or `skills/quality/testing-imports/` if it is a reusable engine upgrade.
