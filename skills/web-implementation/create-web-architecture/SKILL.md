# Skill: Create Web Architecture

## Purpose
Create the architecture plan for a web repo after stack selection.

## Required inputs
- selected web stack
- design baseline or planned design stage
- backend/API plan
- environment strategy

## Steps
1. Define repo structure.
2. Define routing approach.
3. Define component architecture.
4. Define state/API/auth strategy.
5. Define test commands and QA expectations.
6. Update web registries and create a code-agent work packet.
