# Recommend Best Tool for Task

## Purpose

Recommend the best model, tool, or third-party app before doing work that may be better performed outside the current environment.

## Rules

1. Classify the task.
2. Recommend the best tool/environment.
3. Explain why briefly.
4. Offer two paths:
   - proper tool path
   - draft-only current-tool path
5. If the user continues here, label output limitations.
6. If the recommended tool is not integrated, suggest integration-pack creation before activation.

## Examples

- Coding work -> Codex in target repo.
- UI/prototype -> Figma/Figma Make.
- Icons/illustrations -> image generation then Figma review.
- Store screenshot composition -> Figma + image generation support.
- Package/library selection -> research + package decision workflow.
