# Context Backend Cost Review

Free/low-cost options:

- File/GitHub only: free and default
- Local Qdrant via Docker: free, local resources only
- Supabase free tier: useful for small projects, monitor limits
- Qdrant Cloud free/trial tier if available: good for small shared prototypes

Paid tiers may be needed when:

- many projects are indexed
- storage grows
- team access is required
- embedding API usage is high
- uptime/security/backups are needed

AppForge must ask before enabling paid services or paid embedding APIs.
