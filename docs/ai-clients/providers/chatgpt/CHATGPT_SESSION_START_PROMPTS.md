# ChatGPT Session Start Prompts

## Start a new app

```text
Use AppForge. Start a new app from this idea:

[APP IDEA]

Do not create code. First create or update the product brief, ask only critical setup questions, select the current stage, and list which AppForge files should be updated.
```

## Continue current app

```text
Use AppForge. Continue from APPFORGE_PROJECT_STATUS.md. Tell me the current stage, missing inputs, next approval gate, and the next packet or prompt we should create.
```

## Prepare Codex work

```text
Use AppForge. Prepare a Codex implementation packet for [backend/iOS/Android] repo. Include required context files, task scope, acceptance criteria, files likely affected, tests to run, and registry updates required after completion.
```

## Review Figma export

```text
Use AppForge. Review this Figma export/screenshot/design packet. Classify whether it is ready to become the approved design baseline, and list required fixes before frontend implementation.
```

## Change approved UI

```text
Use AppForge. I want to change the approved UI: [CHANGE]. Create a change request, classify impact, identify affected screens/APIs/repos, and tell me which baseline or contract must be updated.
```
