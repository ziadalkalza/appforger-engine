# Document Classification Policy

Document types include:
- meeting_notes
- product_requirements
- design_reference
- backend_reference
- frontend_reference
- qa_report
- release_document
- legal_privacy_reference
- research_note
- external_reference
- imported_document

Lifecycle statuses:
- draft
- candidate
- approved
- current
- superseded
- archived
- external_link_only
- sensitive_restricted

Documents should link to relevant AppForge entities when possible.
