# External Validation Checklist

Before work starts, check:

- [ ] workspace root exists
- [ ] `appforge-engine/` exists
- [ ] `project-control/` exists
- [ ] required project-control files exist
- [ ] required registries exist
- [ ] selected code repos exist
- [ ] selected code repos contain `APPFORGE_POINTER.md`
- [ ] selected code repos contain `AGENTS.md`
- [ ] selected code repos are Git repos if work should persist
- [ ] `design-assets/` exists
- [ ] `exports/` exists
- [ ] `local-only/` exists
- [ ] selected Figma file/frame links are registered when design work is active
- [ ] selected backend/cloud dependencies are registered when backend work is active
- [ ] execution packet target paths are valid before Codex starts
