# Flexible Workflow Audit

Checks v18 stage overrides.

## Must pass

- Figma-less design can be supported through an approved visual baseline.
- HTML sandbox can become a visual baseline only after review and promotion.
- Existing frontend/backend imports require inventory, gap analysis, and approval.
- Backendless apps require a no-backend waiver and impact check.
- Security/privacy/release blockers cannot be silently skipped.
- Imported artifacts cannot bypass documentation, testing, or evidence requirements.
