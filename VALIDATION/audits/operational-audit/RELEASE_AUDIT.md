# Release Audit

Checks v15 release behavior.

## Must pass

- Release candidate is a formal status.
- Required tests must pass unless non-critical tests are explicitly waived.
- Critical blockers require high-risk override if waived.
- Release notes, store/web metadata, screenshots, privacy/checklist placeholders, staged rollout, rollback/hotfix, and post-release monitoring are covered.
- AppForge coordinates release; it does not silently publish.
