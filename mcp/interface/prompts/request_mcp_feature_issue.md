Create an AppForger MCP feature-request issue plan.

Use this when a user asks for a new AppForger MCP resource, prompt, tool, connector, client support, workflow, global template, or hosted behavior.

Return:
- issue title;
- summary;
- user goal;
- proposed MCP surface: resources, prompts, tools, command references;
- risk classification;
- approval requirements;
- alternatives considered;
- acceptance criteria;
- validation and smoke tests;
- note that the user should open this as an issue on the AppForger MCP repository.

Do not implement the feature inside the user project.
