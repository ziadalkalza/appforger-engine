# Storage Decision Matrix

| Artifact | Location | Commit target |
|---|---|---|
| Engine rule/skill/template | `appforge-engine/` | AppForge engine repo |
| Product decision | `project-control/DECISION_LOG.md` | Project-control repo |
| Project state | `project-control/APPFORGE_PROJECT_STATUS.md` | Project-control repo |
| Registry entry | `project-control/REGISTRIES/` | Project-control repo |
| Figma baseline packet | `project-control/BASELINES/` | Project-control repo |
| Codex work packet | `project-control/CODEX_WORK_PACKETS/` | Project-control repo |
| SwiftUI code | `ios/` | iOS repo |
| Compose code | `android/` | Android repo |
| Web code | `web/` | Web repo |
| Backend code | `backend/` | Backend repo |
| Approved logo/icon/image | `design-assets/approved/` | Design-assets repo/storage |
| Pending/generated asset | `design-assets/pending/` | Optional |
| Failed image generation | `local-only/failed-generations/` | Do not commit |
| Raw debug log | `local-only/debug/` | Do not commit |
| QA report | `exports/qa/` or `project-control/QA/` | Commit if it affects decisions/release |
| Screenshot evidence | `exports/screenshots/` | Commit if part of visual QA/release evidence |
