# Trello First Setup

Default board structure: one board per app.

Recommended lists:

- Ideas
- Backlog
- Ready
- In Design
- Backend
- Frontend
- QA
- Blocked
- Done

Recommended labels:

- epic
- story
- design_task
- backend_task
- frontend_task
- qa_task
- bug
- release_task
- documentation_task

Appforger should generate both a manual import/export packet and an API/execution packet.
