# Agent Role Definitions

Default AppForge sub-agent roles:
- Product Agent
- UX/Flow Agent
- Design Agent
- Image Asset Agent
- Backend Agent
- API Contract Agent
- iOS Agent
- Android Agent
- Web Agent
- QA Agent
- Release Agent
- Documentation Agent
- Integration Agent
- Security/Privacy Agent
- Context Backend Agent
- Automation Agent
- AppForge Operator Agent

One person or provider may perform multiple roles. A role does not grant authority to approve baselines, release candidates, integrations, or security-sensitive changes unless assigned in project-control.
