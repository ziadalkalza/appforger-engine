# Non-negotiable Rules

The following cannot be bypassed silently:
- source-of-truth boundaries;
- secrets policy;
- security/privacy review when relevant;
- integration/package approval before use;
- task completion evidence classification;
- conflict/change request handling;
- team-mode PR/review rules;
- release blocker classification;
- production release approval;
- draft/sandbox promotion before canonical use;
- context backend freshness checks when enabled.

High-risk override is possible only when explicitly recorded, approved, and marked with accepted risk. Critical release/security blockers cannot be hidden or silently downgraded.
