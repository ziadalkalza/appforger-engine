# Project Design Override Rules

## Purpose

Project-specific design rules add onto the global design system.

## Priority before Figma approval

1. User-provided design references.
2. Project-specific design system.
3. Global AppForge design system.
4. Platform defaults.

## Priority after Figma approval

Approved Figma baseline is the current visual implementation source of truth.

## Figma does not override

- Accessibility.
- Security.
- Backend/API contracts.
- Platform constraints.
- Legal/privacy rules.
- Release requirements.
