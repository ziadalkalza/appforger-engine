# Figma Workflow

## Role of Figma

Figma can handle:

- UI/UX design.
- Components.
- Design tokens.
- Prototype interactions.
- Animation planning.
- Asset layout.
- Implementation handoff.

## Workflow

```text
UX flows
→ Visual direction
→ Figma prompt/handoff
→ Figma/Figma Make work
→ Review
→ Revisions
→ Design baseline approval
→ Figma-to-Codex handoff
```

## Approval

A Figma file/frame is not implementation truth until approved and registered as a design baseline.

## After approval

If UI changes, create a change request and approve a new design baseline.
