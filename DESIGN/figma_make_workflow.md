# Figma Make Workflow

## Purpose

Use Figma Make to speed up UI/prototype creation.

## Input requirements

- Product summary.
- Screen list.
- UX flows.
- Visual direction.
- Component expectations.
- Interaction expectations.
- Asset list.
- Platform targets.

## Output requirements

- Prototype/screens.
- Components.
- Tokens.
- Animation notes.
- Assumptions.
- Export/import notes.

## Important

Figma Make output must be reviewed. It is not automatically approved.
