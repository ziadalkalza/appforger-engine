# Figma Make Prompt Protocol

Use Figma Make to accelerate design/prototype creation, not to replace AppForge state.

## Prompt structure

```text
Build a high-fidelity prototype for [app/feature].
Use this product context: [brief].
Use this design direction: [visual direction].
Create screens: [screen list].
Include interactions: [interaction list].
Use placeholder data: [data].
Avoid: [constraints].
Output should be clean enough to convert to editable Figma screens/components.
```

After generation, review and refine in Figma. Do not mark as approved until the Figma baseline packet is complete.
