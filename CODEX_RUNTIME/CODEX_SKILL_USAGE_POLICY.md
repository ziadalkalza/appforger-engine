# Codex Skill Usage Policy

Use the relevant `SKILL.md` for every non-trivial task.

Examples:

```text
Backend stack selection -> SKILLS/backend_skills/select_backend_stack/SKILL.md
Supabase backend -> SKILLS/backend_skills/build_supabase_backend/SKILL.md
SwiftUI feature -> SKILLS/ios_skills/implement_swiftui_feature/SKILL.md
Compose feature -> SKILLS/android_skills/implement_compose_feature/SKILL.md
Change request -> SKILLS/governance_skills or change protocol files
```

If a needed skill does not exist, run the engine upgrade protocol or create a pending skill proposal.
