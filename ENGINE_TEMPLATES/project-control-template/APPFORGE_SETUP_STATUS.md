# AppForge Setup Status

```yaml
initialization_status: not_started
questionnaire_version: 1
minimal_gate_complete: false
full_setup_complete: false
pending_user_request: null
pending_critical_questions: []
pending_optional_questions: []
last_updated: TBD
```

## Status meanings

- `not_started`: no onboarding answers saved.
- `in_progress`: onboarding started but minimal gate incomplete.
- `complete_minimal`: required setup complete; non-critical pending questions may remain.
- `complete_full`: all known setup sections complete.
- `needs_update`: project changed and affected sections need re-onboarding.
