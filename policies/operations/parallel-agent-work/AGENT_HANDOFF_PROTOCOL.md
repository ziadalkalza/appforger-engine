# Agent Handoff Protocol

A handoff transfers responsibility from one agent/human/tool to another.

A handoff packet must include:
- source agent
- target agent/role
- related feature/story/task
- source artifacts
- current status
- unresolved questions
- risks
- required next action
- whether reconciliation is required

Cross-brand dependent handoffs should include explicit source-of-truth references and acceptance criteria to avoid hidden logic drift.
