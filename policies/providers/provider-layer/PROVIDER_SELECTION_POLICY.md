# Provider Selection Policy

Provider selection must consider:

- required capabilities
- current provider availability
- project preference
- repo/tool access
- privacy/security impact
- cost
- team access
- output evidence needs
- whether the task is draft, candidate, or production-affecting

If a task requires codebase changes and tests, AppForge should recommend a code agent such as Codex, Claude Code, or an approved local code agent. If the user continues in chat, output must be marked draft-only or unapplied.
