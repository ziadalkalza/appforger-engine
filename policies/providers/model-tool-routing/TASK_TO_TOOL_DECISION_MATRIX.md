# Task to Tool Decision Matrix

| Task | Recommended tool | Why |
|---|---|---|
| product brief, epics, user stories | ChatGPT/AppForge | planning and structured reasoning |
| codebase edits | Codex | repo access, file edits, tests |
| build/test fixes | Codex | can inspect code and run commands |
| high-fidelity UI/prototype | Figma/Figma Make | visual source of truth |
| logo/icon/illustration concepts | image generation | visual asset creation |
| approved screen implementation | Codex with Figma/design packet | implementation from source-of-truth design |
| app store metadata | ChatGPT/AppForge + user review | release copy and checklist |
| advanced animation | Figma/Rive/Lottie tool as appropriate | specialized motion tools |
| payments/maps/messaging | integration pack + Codex | security and external SDK/API work |
