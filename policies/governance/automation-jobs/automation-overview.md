# Automation Overview

Appforger v19 supports automation through controlled job definitions.

Supported execution modes:

- Manual command
- Codex-requested task
- Local script
- Local Git hook
- GitHub Action
- Scheduled GitHub Action
- Future MCP/server job

Automation is not the same as autonomy. Appforger may generate, validate, test, index, and report, but it must stop at approval gates.
