# External Artifact Injection Policy

Existing artifacts may be injected into Appforger, including:
- existing frontend repo;
- existing backend repo;
- Figma file;
- HTML prototype;
- design system;
- documentation set;
- API spec;
- test suite;
- package/library evaluation;
- screenshots or videos.

Injected artifacts do not become canonical automatically. They must be inventoried, reviewed, classified, and either rejected, kept as reference, or promoted into project-control.
