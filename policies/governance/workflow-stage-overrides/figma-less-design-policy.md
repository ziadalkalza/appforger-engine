# Figma-less Design Policy

Figma is recommended for high-fidelity design, but not mandatory.

Allowed Figma-less visual baseline sources:
- HTML sandbox prototype;
- existing frontend implementation;
- screenshot set;
- design system + screen specs;
- image mockups;
- manually written UI specifications.

Before frontend production implementation, there must still be an approved visual source of truth. If Figma is skipped, the substitute baseline must be recorded in VISUAL_BASELINE_REGISTRY.md.
