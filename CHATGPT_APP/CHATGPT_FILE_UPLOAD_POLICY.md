# ChatGPT File Upload Policy

Upload only what the current stage needs.

## Safe to upload

- AppForge Markdown files
- registries
- Figma screenshots/exports
- design tokens
- API contracts
- test reports
- error logs
- small code snippets for review
- generated image options for approval

## Avoid uploading unless needed

- entire app repositories
- secrets
- `.env` files
- keystores/certificates
- service role keys
- large build artifacts
- private customer data

## If code review is needed

Prefer uploading:
- relevant source files only
- build error output
- test output
- implementation packet

Do not use ChatGPT as the primary source repository. GitHub remains the code source of truth.
