# Codex Skill Usage Policy

Use the relevant `SKILL.md` for every non-trivial task.

Examples:

```text
Backend stack selection -> skills/backend-implementation/select-backend-stack/SKILL.md
Supabase backend -> skills/backend-implementation/supabase/build-backend/SKILL.md
SwiftUI feature -> skills/mobile-implementation/implement-swiftui-feature/SKILL.md
Compose feature -> skills/mobile-implementation/implement-compose-feature/SKILL.md
Change request -> skills/governance/governance_skills or change protocol files
```

If a needed skill does not exist, run the engine upgrade protocol or create a pending skill proposal.
