# Draft-Only Current Tool Policy

If the current provider is not suitable for the requested work, Appforger may continue only in a limited mode.

Examples:

- ChatGPT asked to edit repo code → draft patch or code-agent packet only.
- Code agent asked to approve product scope → prepare report, but user/authorized role approves.
- Image generation asked to create final brand asset → candidate asset only until reviewed.

High-risk tasks must never be treated as completed from an unsuitable provider.
