# ChatGPT App Runtime

Use this folder when Appforger is being operated from the ChatGPT app instead of directly inside Codex.

ChatGPT is best for:
- ideation
- requirement shaping
- architecture decisions
- prompt generation
- design direction
- image generation direction
- reviewing Figma exports/screenshots
- creating implementation packets for Codex
- deciding change requests

ChatGPT should not be treated as the production code repo. Production code belongs in the sibling implementation repos.
