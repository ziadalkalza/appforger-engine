# Edge Case Discovery Checklist

Ask for each feature:

- What if required data is missing?
- What if the user is offline?
- What if the API is slow/fails?
- What if the user lacks permission?
- What if duplicate requests happen?
- What if input is too long/short/malformed?
- What if device/browser size is small?
- What if session expires?
- What if external service is unavailable?
- What should be logged, and what must not be logged?
