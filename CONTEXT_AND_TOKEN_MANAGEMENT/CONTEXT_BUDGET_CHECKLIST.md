# Context Budget Checklist

Before loading files, ask:

- Is this a normal task, stage task, or global audit?
- Can a baseline packet replace old raw history?
- Can a stage summary replace long conversation context?
- Which exact registry entries are needed?
- Which repo is actually being edited?
- Is the user-facing `DO_NOT_LOAD` manual unnecessary for this task?
- Can this be handled with one Codex work packet?

If the answer is unclear, start with the operating context and ask for the smallest missing artifact.
