# Add Team Member Workflow

1. Collect name, role(s), GitHub username, Figma/Trello identifiers if relevant.
2. Add member to `TEAM_MEMBER_REGISTRY.md`.
3. Add role assignments to `ROLE_ASSIGNMENT_REGISTRY.md`.
4. Generate a role context pack.
5. Create access request checklist.
6. Record who approved the addition.

AppForge can record access needs, but actual access must be granted in external tools.
