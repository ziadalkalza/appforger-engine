# New App Questionnaire

## Product
- What problem does the app solve?
- Who is the target user?
- What is the first user journey that must work?
- What makes this app different?

## Platforms
- iOS only, Android only, both native, or generic spec-first?
- Should one native platform lead the other?

## Design
- Use AppForge global design system only?
- Import design system like Impeccable or a Figma kit?
- Generate assets with Image 2?
- Existing logos/brand files?

## Backend
- Supabase or FastAPI + PostgreSQL?
- Accounts/auth required?
- Realtime, uploads, notifications, payments, maps, offline mode?

## Repos
- GitHub organization/user.
- Repo naming convention.
- Public/private.

## Manual setup
- Figma file exists?
- Supabase project exists?
- DigitalOcean/Render account exists?
