# SQLite Team Operations Profile

SQLite is a supported lightweight Team Operations Backend option for local, small, or simple deployments.

It can be used locally, inside a controlled shared environment, or as part of a lightweight hosted service. It is not automatically equivalent to a cloud multi-user backend; concurrency, backup, and access controls depend on how it is hosted.

Use it for:
- local/offline operation
- small teams with controlled access
- prototypes
- low-cost deployments
