# Context Backend Cost Review

Free/low-cost options:

- File/source-control app only: free and default
- Local vector database via Docker: free, local resources only
- SQL/vector storage free tier: useful for small projects, monitor limits
- Vector database cloud free/trial tier if available: good for small shared prototypes

Paid tiers may be needed when:

- many projects are indexed
- storage grows
- team access is required
- embedding API usage is high
- uptime/security/backups are needed

Appforger must ask before enabling paid services or paid embedding APIs.
