# Retrieval Pack Rules

A retrieval pack is a focused context bundle for a task/role.

Every retrieved item must include:

- source path/link
- approval status
- current/superseded status
- baseline version
- last indexed timestamp
- retrieval score

Types:

- frontend_context_pack
- backend_context_pack
- design_context_pack
- qa_context_pack
- release_context_pack
- integration_context_pack
- user_story_context_pack
