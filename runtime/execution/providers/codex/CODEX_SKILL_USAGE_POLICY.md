# Codex Skill Usage Policy

Use the relevant `SKILL.md` for every non-trivial task.

Examples:

```text
Backend stack selection -> skills/implementation/backend/generic/select_backend_stack/SKILL.md
Supabase backend -> skills/implementation/backend/apps/supabase/build_backend/SKILL.md
SwiftUI feature -> skills/implementation/ios_skills/implement_swiftui_feature/SKILL.md
Compose feature -> skills/implementation/android_skills/implement_compose_feature/SKILL.md
Change request -> skills/governance/governance_skills or change protocol files
```

If a needed skill does not exist, run the engine upgrade protocol or create a pending skill proposal.
