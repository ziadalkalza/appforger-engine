# Generic Spec-First Rules

## Meaning

Generic spec-first means platform-neutral product/design/API planning.

It does not mean Flutter.

## Outputs

- Product brief.
- Shared feature specs.
- Shared screen specs.
- UX flows.
- Figma prototype.
- Backend/API plan.
- Native adaptation plans for iOS and Android.

## Forbidden

- Flutter implementation.
- React Native implementation.
- Wrapper framework frontend.
- Prototype code pretending to be final native code.

## Why

Figma handles prototypes. Native apps are implemented later using SwiftUI and/or Kotlin Compose.
