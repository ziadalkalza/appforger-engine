# New Project Questionnaire

Use this when the user starts from scratch.

Required minimal flow:

1. Project name.
2. Target platforms.
3. Single-user or team.
4. Workflow preference.
5. Design workflow.
6. Backend needed or not sure.
7. Tool consistency mode + provider preference.
8. Docs storage.
9. Context backend mode.
10. Automation level.
11. Public release expected.
12. Likely integration needs.
