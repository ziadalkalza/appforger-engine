# Design Workflow Questionnaire

Ask:

1. Is UI/design needed now?
2. Preferred design route:
   - Figma first
   - HTML sandbox first
   - existing frontend/imported design
   - screenshots/reference design
   - no UI yet
3. Can HTML sandbox or imported frontend become visual baseline after review?
4. Will image generation be used for icons, illustrations, mascots, app screenshots, or design assets?

The answer controls Figma workflows, draft/sandbox workflows, visual baseline rules, and provider routing.
