# Release and Integration Questionnaire

Initial onboarding should only ask whether public release is expected.

```text
Public release expected? yes / no / not sure
```

Do not run full release planning during onboarding.

Ask likely integration needs only:

```text
payments/subscriptions
maps/location
notifications
analytics
email/SMS
storage/files
none
not sure
```

Specific integration setup happens later through integration-pack workflows.
