# Native Adaptation Rules

## Purpose

Adapt shared specs and approved Figma into SwiftUI and Kotlin Compose.

## Adaptation checklist

- Navigation differences.
- Platform components.
- Safe areas/system bars.
- Back behavior.
- Keyboard behavior.
- Accessibility.
- Animation differences.
- Asset export sizes.
- API/client differences.

## Rule

Native adaptation should preserve product behavior and visual intent while respecting platform conventions.
