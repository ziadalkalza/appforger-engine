# Image Generation Review Workflow

## Use cases

- Logos.
- Mascots.
- Icons.
- Onboarding illustrations.
- Empty states.
- Subscription illustrations.
- App store screenshots/marketing visuals.

## Workflow

```text
Prompt
→ Generate variations
→ Review
→ Revise
→ Approve
→ Import to Figma
→ Export to app repos
```

## Review checklist

- Fits product.
- Fits visual direction.
- Works at mobile sizes.
- Does not rely on copyrighted/pop-culture references unless cleared.
- Is exportable.
- Does not conflict with Figma baseline.
