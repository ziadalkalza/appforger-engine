# Animation Asset Workflow

## Animation sources

- Figma prototypes.
- Figma component variants.
- Lottie/Rive if approved through engine upgrade or project rule.
- Native animations implemented in SwiftUI/Compose.

## Required documentation

- Trigger.
- Duration.
- Easing.
- Start/end states.
- Platform differences.
- Accessibility fallback for reduced motion.
