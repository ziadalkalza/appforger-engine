# Appforger v11 Capability Boundary

v11 is a file-based app-building engine. It can coordinate and generate artifacts, but it is not an autonomous cloud service.

## Included

- mobile and web tracks,
- design/Figma/image workflows,
- project-control separation,
- remote/multi-device rules,
- integration-pack schema and research protocol,
- connector/MCP/skill lifecycle tracking,
- plan mode and uncertainty rules,
- task completion/evidence rules,
- validators and bootstrap scripts.

## Excluded by design for now

- live Appforger MCP server,
- centralized SQL/RAG storage,
- full multi-user permission dashboard,
- automatic connector installation/authorization,
- background sync jobs,
- automatic cloud account creation.
