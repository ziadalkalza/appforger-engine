# Draft to Baseline Rules

A draft can become a baseline only after review and approval.

## Visual draft

Requires:

- Figma or visual artifact link;
- affected screens/components;
- design review;
- accessibility/platform check;
- user approval.

## API/backend draft

Requires:

- endpoint/data model proposal;
- backend/security review;
- affected frontend stories;
- test plan;
- user/API owner approval.

## Package draft

Requires:

- package evaluation;
- license/security review;
- package decision approval.
