# Draft to Codex Packet Rules

Draft code or prototypes must be converted into a real execution packet before production implementation.

A promoted execution packet must include:

```text
source draft ID
approved target behavior
source of truth baseline
target repo
files to inspect
files to edit
forbidden paths
tests to run
done report requirements
```

Never tell Codex to "just use the draft" without translating it into a proper task packet.
