# Task Completion Checklist

- [ ] Output created.
- [ ] No app code was written inside AppForge engine folder.
- [ ] Relevant baseline/registry updated.
- [ ] If output came from external tool, artifact import completed.
- [ ] Tests/checks run or manual test instructions provided.
- [ ] Open questions captured.
- [ ] Next task suggested.
## Evidence-Based Closeout

Before setting a task to `completed_verified`, consult:

```text
workflows/operations/task-completion-evidence/TASK_CLOSEOUT_CHECKLIST.md
workflows/operations/task-completion-evidence/COMPLETION_EVIDENCE_MATRIX.md
```

If the user only saved partial work, use `in_progress_committed`.
If the user closed work without evidence, use `manually_closed` or `user_marked_done_needs_evidence`.
