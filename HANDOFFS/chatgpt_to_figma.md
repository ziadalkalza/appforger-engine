# ChatGPT to Figma Handoff

## Purpose

Move approved product/UX/visual context into Figma or Figma Make.

## Include

- Product summary.
- Target users.
- Screen list.
- User flows.
- Visual direction.
- Asset list.
- Interaction/animation notes.
- Platform targets.
- Constraints.

## Output expected from Figma

- Figma file/frame links.
- Components.
- Prototype interactions.
- Asset exports.
- Animation notes.
- Assumptions.
