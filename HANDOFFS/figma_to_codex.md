# Figma to Codex Handoff

## Purpose

Move approved Figma baseline into native implementation tasks.

## Required

- Design baseline version.
- Figma frame/component links.
- Screen IDs.
- Component names.
- Design tokens.
- Asset export list.
- Motion notes.
- Platform differences.

## Rule

Codex must implement from this handoff and must not invent visual UI.
