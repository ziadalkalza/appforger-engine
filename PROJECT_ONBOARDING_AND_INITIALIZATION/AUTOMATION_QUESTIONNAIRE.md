# Automation Questionnaire

Ask minimal automation level during setup.

Options:

```text
0. Manual only
1. Draft generation only
2. Local validation/reporting
3. Agent execution after approval
4. GitHub Actions enabled
5. Future runner/server later
```

Recommended default for new projects: Level 2.

Follow-up questions:
- Can AppForge generate next-step packets automatically?
- Can AppForge install local Git hooks after approval?
- Can AppForge create GitHub Actions workflow files after approval?
- Can agents update registries if packets allow it?
