# Team Mode Questionnaire

Ask this only if team mode is selected.

Required:

1. Team members and roles.
2. Named approvers for product, design, API/backend, QA, release.
3. Should project-control changes require PRs? Recommended: yes.
4. Should Team Operations Backend be enabled?
5. Preferred team operations backend: Supabase, FastAPI/Postgres, SQLite, later.
6. Who can manage secret metadata?
7. Who can approve high-risk overrides?

Optional role-specific questions may be answered later by the responsible role.
