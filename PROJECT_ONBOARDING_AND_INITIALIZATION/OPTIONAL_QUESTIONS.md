# Optional Questions

Optional questions are asked only when relevant or when the user chooses full setup.

Examples:

- Exact backend provider if backend is still uncertain.
- App Store / Google Play / web publishing details.
- Specific integrations.
- Detailed automation schedules.
- Advanced context backend provider.
- Team Operations Backend provider.
- Full team member list.
- Security/privacy reviewer names.
- Package preferences.

Optional answers are stored as `unknown_pending` until needed.
