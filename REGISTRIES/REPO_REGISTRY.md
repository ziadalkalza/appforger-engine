# Repository Registry

AppForge assumes separate repos by default.

| Repo role | Repo URL | Branch policy | Owner | Status | Connected artifacts |
|---|---|---|---|---|---|
| Engine docs | | | | | |

| Android | | | | | |
| Design assets | | | | | |

## Connection rules

- Backend repo owns API implementation and migrations.
- iOS and Android repos consume approved API contracts.
- Figma owns approved visual baseline.
- Engine docs repo owns context, rules, prompts, registries, and change requests.
