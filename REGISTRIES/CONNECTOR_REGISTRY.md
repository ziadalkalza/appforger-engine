# Connector Registry

## First-class connectors

| Connector | Purpose | Status | Notes |
|---|---|---|---|
| GitHub | Code repos, issues, PRs | approved | primary code connector |
| Figma | Design, prototype, assets, MCP context | approved | primary design connector |
| Supabase | Backend projects, DB/auth/storage/edge functions | approved | default backend option |
| Google Drive | Docs/assets/source files | approved | handoff/source repository |
