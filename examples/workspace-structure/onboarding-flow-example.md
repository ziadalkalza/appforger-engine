# Onboarding Flow Example

User request before initialization:

```text
Build the login screen.
```

Appforger response:

```text
Before I can route this task correctly, the project needs minimal initialization.
I will ask the required setup questions, save the answers, generate active rules, then return to: Build the login screen.
```

Minimal answers are saved to:

```text
project-control/APPFORGER_PROJECT_PROFILE.md
project-control/APPFORGER_ONBOARDING_ANSWERS.md
project-control/APPFORGER_ACTIVE_RULES.md
project-control/APPFORGER_SETUP_STATUS.md
project-control/APPFORGER_PROJECT.yaml
```

Then Appforger resumes the paused task and recommends the correct workflow/provider.
