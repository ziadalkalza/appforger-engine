# Team Collaboration Example

This example shows how a small team exists and operates in Appforger. It is not a full app example.

Team:
- Ziad: Appforger Operator + Product Owner
- Sara: UI/UX Designer
- Omar: Backend Developer
- Nour: Mobile Developer
- Lina: QA Tester

Flow:
1. Product owner approves USER_STORY_AUTH_001.
2. Designer creates DESIGN_BASELINE_AUTH_V1.
3. Backend developer creates API_BASELINE_AUTH_V1.
4. Mobile developer implements TASK-IOS-LOGIN-001.
5. QA tester verifies test evidence.
6. Release manager/project owner signs off release readiness.
