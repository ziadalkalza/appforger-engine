# Draft Sandbox Example

This example shows a simple HTML layout exploration for a mobile onboarding flow.

It is intentionally not a real frontend implementation.

## Flow

```text
Create DRAFT-ONBOARDING-001
→ store raw HTML in local-only/drafts/html-prototypes/onboarding-v1/
→ export screenshots to exports/drafts/DRAFT-ONBOARDING-001/
→ review
→ promote into Figma prompt or screen spec only if approved
```

## Example draft record

```yaml
draft_id: DRAFT-ONBOARDING-001
title: Onboarding layout exploration
type: html_prototype
status: active
linked_story: STORY-ONBOARDING-001
location: local-only/drafts/html-prototypes/onboarding-v1/
affects_project_truth: false
promotion_required: true
expiry_review_date: 2026-06-08
```
