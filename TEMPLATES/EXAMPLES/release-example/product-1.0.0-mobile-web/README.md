# Release Example — Product 1.0.0 Mobile + Web

This compact example shows how AppForge tracks a grouped product release with platform-specific releases.

```text
Product Release: 1.0.0
Release Candidate: 1.0.0-rc.1
Platforms:
- backend 1.0.0
- iOS 1.0.0
- Android 1.0.0
- web 1.0.0
```

This is an example only. It is not active project state.
