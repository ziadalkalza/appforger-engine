# Testing Skill Stack Mapping

Imported testing skills must be mapped to actual project stacks.

Examples:

- Generic web testing guidance → Next.js/React/Vite commands and Playwright/Cypress decisions.
- Generic mobile QA checklist → SwiftUI and Compose screen states/device checks.
- Generic API testing rules → FastAPI/Supabase endpoint and auth/RLS checks.
- Accessibility guide → platform-specific iOS/Android/web accessibility checks.
