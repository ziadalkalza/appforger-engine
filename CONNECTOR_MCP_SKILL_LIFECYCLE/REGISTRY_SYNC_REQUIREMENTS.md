# Registry Sync Requirements

When a connector, MCP, skill, or integration changes state, update the relevant registry:

- `REGISTRIES/CONNECTOR_LIFECYCLE_REGISTRY.md`
- `REGISTRIES/MCP_LIFECYCLE_REGISTRY.md`
- `REGISTRIES/SKILL_LIFECYCLE_REGISTRY.md`
- `REGISTRIES/CONNECTOR_REGISTRY.md`
- `REGISTRIES/MCP_REGISTRY.md`
- `COST_AND_DEPENDENCIES/SERVICE_DEPENDENCY_REGISTRY.md`

A capability is not considered active until the registry says active and the validation evidence is recorded.
