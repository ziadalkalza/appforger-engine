## Required screenshots/exports

Ask the user or connector to provide:

```text
- full-page screenshots of relevant frames
- component screenshots
- token/style screenshots or exports
- component names
- variant names
- color styles
- text styles
- spacing/layout rules
- prototype links or interaction notes
```

## Implementation warning

A Figma UI kit is not automatically production-ready. It must be adapted for:

- platform navigation behavior
- accessibility
- responsive layout
- real data states
- loading/error/empty states
- native controls where appropriate
