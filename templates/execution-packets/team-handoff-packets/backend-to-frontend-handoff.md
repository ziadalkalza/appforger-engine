# Backend to Frontend Handoff

Use this packet to transfer responsibility between roles.

Required fields:
- handoff_id
- from_role
- to_role
- related feature/story/task
- source artifacts
- approved baselines
- unresolved questions
- known risks
- required next action
- required approval/evidence

Must include API baseline, endpoint list, auth rules, error response rules, mock data, and known limitations.
