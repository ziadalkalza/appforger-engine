# Design Review Prompt

```text
Review this design against Appforger rules.

Check:
1. Does it match the product and target users?
2. Does it match the current visual direction?
3. Are the screens clear and not cluttered?
4. Are components reusable?
5. Are there obvious accessibility issues?
6. Are loading/empty/error states covered?
7. Are assets ready for native implementation?
8. What should change before approval?

Classify:
- approve
- approve with minor fixes
- needs revision
- reject
```
