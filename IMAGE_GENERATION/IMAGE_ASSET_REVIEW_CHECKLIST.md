# Image Asset Review Checklist

Mark every generated asset before importing to Figma.

```text
[ ] Matches product concept
[ ] Matches target audience
[ ] Matches approved or proposed design direction
[ ] Works at intended size
[ ] Does not rely on copyrighted/pop-culture references
[ ] Does not look like a known brand mark
[ ] Has clear background/export needs
[ ] Has variants if needed
[ ] User approved
[ ] Imported into Figma or design-assets repo
[ ] DESIGN_ASSET_REGISTRY.md updated
```
