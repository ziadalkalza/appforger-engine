# Figma Draft Workflow

Figma draft work is allowed. It is not an approved design baseline.

## Figma draft states

```text
figma_draft
candidate_design_baseline
approved_design_baseline
discarded
```

## Rules

- Figma draft frames must be clearly named with `DRAFT` or a draft ID.
- Do not let Codex implement from a Figma draft unless the task is explicitly mock-only or draft-only.
- To promote, create a design baseline packet and review it through AppForge.
