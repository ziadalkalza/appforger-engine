# Discard Draft Policy

Discarding a draft should not erase useful learning.

## Discard flow

```text
mark draft as discarded
record reason
optionally keep short summary
remove or ignore raw draft files
prevent future agents from treating it as active
```

## Reasons

- visually poor;
- technically unsuitable;
- package too risky;
- conflicts with approved architecture;
- user no longer wants it;
- superseded by another draft.
