# Draft Review Checklist

Before promoting a draft, check:

```text
[ ] What problem was this draft exploring?
[ ] Which epic/feature/story does it relate to?
[ ] Is the draft visual, code, backend, package, asset, or mixed?
[ ] Does it conflict with approved baselines?
[ ] Does it require user approval?
[ ] Does it require tests/build evidence?
[ ] Does it require package/security/license review?
[ ] What exact artifact should be promoted?
[ ] What should be discarded?
[ ] Which registries must be updated?
```
