# Release Evidence Matrix

Evidence types:

| Evidence | Required For | Location |
|---|---|---|
| Test report | all releases | exports/qa/ or project-control |
| Visual QA | UI releases | exports/screenshots/ |
| Store metadata | mobile | project-control/RELEASE_DOCS/ |
| Web deployment checklist | web | project-control/RELEASE_DOCS/web-publishing/ |
| Production env check | backend/web/mobile integration | project-control/REGISTRIES/DEPLOYMENT_REGISTRY.md |
| Approval record | final release | project-control/REGISTRIES/RELEASE_EVIDENCE_REGISTRY.md |
